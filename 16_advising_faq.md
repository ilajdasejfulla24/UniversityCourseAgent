# Academic Advising FAQ
Students should select courses by checking prerequisites, graduation requirements, interests, weekly workload, and career goals.
If a prerequisite is missing, students should normally take the prerequisite first or ask the instructor for written permission.
For a balanced semester, combine one high-workload course with medium-workload courses and one elective or project course.
Course availability changes each semester and must be checked against the current timetable or registration system.
